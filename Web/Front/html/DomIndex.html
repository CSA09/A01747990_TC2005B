Preguntas

  1. ¿Qué es el DOM?

    2. Chat GPT indica que DOM es una representación estructurada y jerárquica de un documento HTML, XML o XHTML, en la que cada elemento del documento se organiza de acuerdo a una estructura de árbol en relaciones padre-hijo, hermano-hermano entre otros. También indica que esta interfaz permite que los programadores puedan acceder y modifcar de manera dinámica los elementos del documento mediante diferentes métodos

  2. ¿Qué es un nodo?

    3. Chat GPT indica que un nodo en el contexto del DOM, un nodo es n nodo es un objeto que representa un elemento, atributo, texto o comentario en la estructura jerárquica de un documento. Cada nodo tiene un tipo y puede tener relaciones con otros nodos.

  3. ¿Cómo se accede a un elemento del DOM?

    4. Según Chat GPT, para acceder a un elemento, se utilizan diferentes métodos y propiedades disponibles en JavaScript, como por su ID, etiqueta, clase, etc.

  4. ¿Cómo se modifica un elemento del DOM?

    5. De acuerdo con Chat GPT, los elementos del DOM se pueden modificar de acuerdo a su contenido, atributos, estilos y clases, esto mediante distintos comandos en javascript

  5. ¿Cuál es la diferencia entre getElementby* y querySelector*?

    6. La principal diferencia entre ambos es la forma en la que seleccionan y devuelven elementos del DOM, getElementby* seleccionan elementos basándose en propiedades específicas, mientras que querySelector* utiliza selectores CSS para seleccionar elementos para devolver el primer elemento que coincide con el selector.

  6. ¿Qué es un evento?

    7. Es una acción o suceso que ocurre en un elemento del documento HTML y que puede ser detectado y manejado mediante programación. Los eventos en el DOM son acciones que pueden ser desencadenadas por el usuario, el navegador o el propio documento, y permiten que el código JavaScript responda a estas interacciones. Ej: Clic de ratón, Cambio de valor

  7. ¿Qué es un listener?

    8. Es una función o método que se utiliza para detectar y responder a eventos que ocurren en los elementos del DOM. Los listeners en el DOM permiten que un código JavaScript responda a las interacciones del usuario, cambios en el estado de los elementos o cualquier otro evento específico.

  8. ¿Qué es un callback? **

    9. Es una función que se pasa como argumento a otra función y que se invoca en un momento posterior. En otras palabras, es una función que se "llama de vuelta" más adelante en el flujo del programa.

  9. ¿Qué es un objeto de evento?

    10. Un objeto de evento (event object) es un objeto que contiene información sobre un evento específico que ha ocurrido. Los objetos de evento se utilizan en muchos lenguajes de programación y entornos, incluyendo el DOM, para proporcionar detalles sobre un evento y permitir que el código responda en consecuencia. Cuando ocurre un evento, el navegador o el entorno de ejecución genera un objeto de evento que contiene información relevante sobre el evento en particular. Este objeto se pasa como argumento a la función de manejo de eventos (listener/callback) asociada con el evento.

¿Cuándo y para qué usarías ChatGPT y cuándo Copilot?

Usaria ChatGPT para cualquier optimización de procesos así como ayuda en el pensamiento lógico en la resolución de problemas, mientras que a Copilot lo usaría para la optimización de código en particular, esto debido a las funciones que ofrece.

Ventajas de usar AI para programar

Desventajas de usar AI para programar

Muchas veces los métodos que usa la IA son erróneos, y cuando no lo son, si no se tiene cuidado, el uso excesivo y sin cuidado de la IA puede privar al usuario del desarrollo de procesos cognitivos para la resolución de problemas.
