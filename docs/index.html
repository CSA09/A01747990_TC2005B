HTML Cheat Sheet

<a>

The <a> tag defines a hyperlink, which is used to link from one page to another.

Example: <a href="https://www.example.com">Example</a>

<p>

The <p> tag defines a paragraph.

Example: <p>This is a paragraph.</p>

<h1>, <h2>, ..., <h6>

The <h1>, <h2>, ..., <h6> tags define headings of different sizes.

Example: <h1>This is a heading</h1>

<div>

The <div> tag defines a section of a document, which can be styled and manipulated with CSS and JavaScript.

Example: <div>This is a section of the document.</div>

<span>

The <span> tag defines a small section of a document, which can be styled and manipulated with CSS and JavaScript.

Example: <span>This is a small section of the document.</span>

<strong>

The <strong> tag defines important text, which is typically displayed in bold.

Example: <strong>This text is important.</strong>

<i>

The <i> tag defines text that is in an alternative voice or mood, or to indicate a technical term, a phrase from another language, or a thought.

Example: <i>This text is in an alternative voice.</i>

<u>

The <u> tag defines underlined text.

Example: <u>This text is underlined.</u>

<b>

The <b> tag defines text that should be stylistically offset from normal text, typically by making it bold.

Example: <b>This text is stylistically offset.</b>

<strike>

The <strike> tag defines text that should be crossed out.

Example: <strike>This text is crossed out.</strike>

<small>

The <small> tag defines smaller text, which is typically used for disclaimers or legal notices.

Example: <small>This is smaller text.</small>
