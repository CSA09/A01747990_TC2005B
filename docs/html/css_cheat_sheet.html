Border

El borde define un borde alrededor del elemento. Puedes especificar el ancho, estilo y color del borde usando la propiedad CSS border.

Border left

La propiedad border-left define el borde del lado izquierdo del elemento.

Border right

La propiedad border-right define el borde del lado derecho del elemento.

Border top

La propiedad border-top define el borde del lado superior del elemento.

Border bottom

La propiedad border bottom define el borde del lado inferior del elemento

Border radius

Border radius define el radio de la curva de las esquinas de un contenedor.

Background image

Backgorund image coloca una imagen al fondo del elemento defindiendo sus dimensiones en css

Color

Color define el color general del texto dentro del elemento

Font

Font establece todas las diferentes propiedades de la fuente de un elemento.

Font family

Font family especifica una lista priorizada de uno o más nombres de familia de fuentes y/o nombres de familia genéricos para el elemento seleccionado.

font-style-weight

Define estilo de fuente (italic, normal o oblique) y el grosor de la letra

Width and height

Width and height define las dimensiones del elemento contenedor

list-style

Permite establecer todas propiedades de una lista a la vez

Marging and padding

Define los margenes del elemento respecto a otro, así como los márgenes interiores del elemento

Overflow

Establece el comportamiento deseado para el desbordamiento de un elemento, es decir, cuando el contenido de un elemento es demasiado grande para caber en su contexto de formato de bloque, en ambas direcciones.

Display

Display

Establece si un elemento se trata como un bloque o un elemento en línea y el diseño utilizado para sus elementos secundarios, como el flow layout, grid o flex.

Position

Define como se posiciona un elemento en el documento

Filter

Aplica efectos gráficos como desenfoque o cambio de color a un elemento
