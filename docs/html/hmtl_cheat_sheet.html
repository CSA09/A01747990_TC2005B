HTML Cheat Sheet

<a>

El tag <a> define un hipervínculo, el cual se utiliza para enlazar de una página a otra.

Ejemplo: <a href="https://www.ejemplo.com">Ejemplo</a>

<p>

El tag <p> define un párrafo.

Ejemplo: <p>Este es un párrafo.</p>

<h1>, <h2>, ..., <h6>

Los tags <h1>, <h2>, ..., <h6> definen encabezados de diferentes tamaños.

Ejemplo: <h1>Este es un encabezado</h1>

<div>

El tag <div> define una sección de un documento, la cual puede ser estilizada y manipulada con CSS y JavaScript.

Ejemplo: <div>Esta es una sección del documento.</div>

<span>

El tag <span> define una pequeña sección de un documento, la cual puede ser estilizada y manipulada con CSS y JavaScript.

Ejemplo: <span>Esta es una pequeña sección del documento.</span>

<strong>

El tag <strong> define texto importante, el cual se muestra típicamente en negrita.

Ejemplo: <strong>Este texto es importante.</strong>

<i>

El tag <i> define texto que está en una voz o estado alternativo, o para indicar un término técnico, una frase de otro idioma, o un pensamiento.

Ejemplo: <i>Este texto está en una voz alternativa.</i>

<u>

El tag <u> define texto subrayado.

Ejemplo: <u>Este texto está subrayado.</u>

<b>

El tag <b> define texto que debería ser estilísticamente diferente al texto normal, típicamente haciéndolo negrita.

Ejemplo: <b>Este texto está estilísticamente diferente.</b>

<strike>

El tag <strike> define texto que debería ser tachado.

Ejemplo: <strike>Este texto está tachado.</strike>

<small>

El tag <small> define un texto más pequeño que el texto normal.

Ejemplo: <small>Este texto es más pequeño que el texto normal.</small>

<em>

El tag <em> define un texto que se debe enfatizar, normalmente se muestra en cursiva.

Ejemplo: <em>Este texto está enfatizado.</em>

<header>

El tag <header> define un encabezado para una sección o página.

Ejemplo: <header>Este es un encabezado para una sección.</header>

<section>

El tag <section> define una sección de un documento.

Ejemplo: <section>Esta es una sección del documento.</section>

<article>

El tag <article> define un contenido independiente y autocontenido en un documento.

Ejemplo: <article>Este es un artículo independiente.</article>

<aside>

El tag <aside> define un contenido relacionado, pero separado, del contenido principal de un documento.

Ejemplo: <aside>Este es un contenido relacionado, pero separado, del contenido principal.</aside>

<footer>

El tag <footer> define un pie de página para una sección o página.

Ejemplo: <footer>Este es un pie de página para una sección.</footer>

<nav>

El tag <nav> define una sección de navegación en un documento.

Ejemplo: <nav>Esta es una sección de navegación.</nav>

<img>

El tag <img> define una imagen en un documento.

<br>

El tag <br> define un salto de línea.

Ejemplo: Este es un texto que está en la primera línea.<br>Este es un texto que está en la segunda línea.

<hr>

El tag <hr> define una línea horizontal en un documento.

Ejemplo: <hr>

<form>

El tag <form> define un formulario para la entrada de datos del usuario.

Ejemplo: <<form> action="procesar.php" method="post"> <<label> for="nombre">Nombre:</label> <<input> type="text" id="nombre" name="nombre">

<table>

El tag <table> Crea una tabla especificando filas "<tr>" y columnas "<td>"".

Ejemplo: <table> <tr> <th>Nombre</td> <th>Edad</td> </tr> <tr> <td>María</td> <td>25</td> </tr> <tr> <td>Juan</td> <td>30</td> </tr> </table> Que equivale a:

Nombre Edad
María 25
Juan 30

<ul> y <ol>

Los tags <ul> y <ol> Sirven para crear listas no ordenadas y ordenadas (numeradas secuencialmente) respectivamente.

Ejemplo: <ul> <li>Manzanas<li> <li>Plátanos<li> <li>Naranjas<li> </ul> <ol> <li>Hacer tarea 1<li> <li>Hacer tarea 2<li> <li>Hacer tarea 3<li> </ol>
